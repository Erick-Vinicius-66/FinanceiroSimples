💰 Sistema de Gestão Financeira Pessoal
Este projeto é uma aplicação web desenvolvida em C# com ASP.NET Core MVC, focada no controle de receitas e despesas. O objetivo é permitir que usuários gerenciem seu fluxo de caixa, visualizando o saldo total e organizando suas movimentações financeiras de forma intuitiva.

🚀 Tecnologias Utilizadas
Linguagem: C# (.NET 9/10)

Framework: ASP.NET Core MVC

Interface: HTML5, CSS

IDE: Visual Studio

Banco de Dados: Entity Framework Core (SQL Server ou SQLite)

⚙️ Funcionalidades
Dashboard Financeiro: Visualização do saldo total, soma de receitas e soma de despesas.

Gestão de Receitas: Lançar novas receitas e excluir entradas incorretas.

Gestão de Despesas: Adicionar novas despesas que impactam diretamente no seu saldo e exclusão.

Fluxo de Caixa: Cálculo automático do saldo disponível com base nas movimentações.
